# 최단 경로 길이 관련 코딩테스트 문제 유형 조사

코딩테스트에서 최단 경로를 구하는 문제는 매우 빈번하게 출제됩니다. 이 중에서도 BFS(Breadth-First Search)를 이용하여 간단하게 구현할 수 있는 유형부터 시작하여 점차 복잡한 문제로 발전시킬 수 있습니다.

## 1. 최단 거리의 길이를 구하는 문제
기본 적인 BFS로 풀 수 있는 문제. 상하좌우를 돌면서 방문을 체크하고 방문하지 않은 곳이면 방문한다. 큐 자료구조를 사용하며 방문할 노드를 큐에 넣어 순차적으로 처리하도록 한다.

### 관련 문제 링크
백준 2178번 미로 탐색 : https://techbrad.tistory.com/110

## 2. 최단 거리의 총 개수를 구하는 문제
BFS로 풀 수 있지만 DP로도 해결할 수 있다.

### 관련 문제 링크
프로그래머스 등굣길 탐색 : https://techbrad.tistory.com/114


## 2. 학습을 통해 추가할 내용

- **가중치가 있는 그래프에서의 최단 경로 문제**: 다익스트라 알고리즘 활용.
- **음의 가중치가 있는 그래프에서의 최단 경로 문제**: 벨만-포드 알고리즘 활용.
- **플로이드-워셜 알고리즘을 이용한 모든 쌍 최단 경로 문제**.
- **특정 조건이 추가된 BFS 문제**: 이동할 수 있는 방향이 제한되거나 특정한 조건을 만족해야 하는 경우.
